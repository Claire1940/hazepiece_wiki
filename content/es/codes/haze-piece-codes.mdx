Que un código falle no siempre significa que la recompensa haya desaparecido. En muchos casos, el problema es un error de menú, un espacio al final o una entrada vencida que permaneció demasiado tiempo en una lista copiada. Usa la tabla de abajo para diagnosticar el problema rápidamente.

<Callout type="warning" title="Patrón de fallo común">
Si un código parece válido pero no hace nada, vuelve a abrir la ventana de canje, copia el código otra vez y pruébalo en una sesión de servidor nuevo.
</Callout>

| Causa del fallo | Solución rápida | Cuándo ocurre |

| Error de escritura o espacio extra | Pega otra vez desde una fuente limpia | Escritura manual, teclados móviles |
| Código vencido | Elimínalo y prueba uno actual | Publicaciones antiguas y listas desactualizadas |
| Ruta de menú incorrecta | Usa el icono del cofre del tesoro o del regalo | Jugadores nuevos y cambios de interfaz |
| Ya canjeado | Pasa a otro código | Intentos duplicados |
| Retraso de actualización | Vuelve a entrar en un servidor nuevo | Justo después de salir un parche |
