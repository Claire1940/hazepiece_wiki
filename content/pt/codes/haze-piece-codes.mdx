<Section icon={Target} title="Como Resgatar Códigos de Haze Piece">

</Callout>

<Section icon={Shield} title="Por Que os Códigos de Haze Piece Falham e Como Corrigir">

Uma falha no código nem sempre significa que a recompensa acabou. Em muitos casos, o problema é um erro de menu, um espaço no final ou uma entrada expirada que ficou tempo demais em uma lista copiada. Use a tabela abaixo para diagnosticar o problema rapidamente.

<Callout type="warning" title="Padrão de Falha Comum">
Se um código parece válido, mas não faz nada, reabra a janela de resgate, copie o código novamente e teste em uma sessão de servidor nova.
</Callout>

| Causa da Falha | Correção Rápida | Quando Acontece |
|---|---|---|
| Erro de digitação ou espaço extra | Cole novamente de uma fonte limpa | Digitação manual, teclados de celular |
| Código expirado | Remova e tente um atual | Posts antigos e listas desatualizadas |
| Caminho de menu errado | Use o ícone de Baú do Tesouro ou Presente | Jogadores novos e mudanças na UI |
| Já resgatado | Passe para outro código | Tentativas duplicadas |
| Atraso na atualização | Entre novamente em um servidor novo | Logo após o lançamento do patch |

<Section icon={Swords} title="Melhor Forma de Gastar as Recompensas de Haze Piece">

A melhor estratégia com códigos não é resgatar tudo de uma vez. É combinar cada recompensa com uma meta de curto prazo. Os boosts de EXP são a recompensa de maior ritmo, porque apoiam diretamente o ganho de nível, enquanto cash e gems são melhores guardados para o próximo gargalo que realmente atrasa seu progresso.

<Callout type="info" title="Regra do Melhor Valor">
Se uma recompensa economiza tempo, use-a antes de começar o farm. Se uma recompensa muda sua build, use-a apenas quando estiver pronto para assumir essa decisão.
</Callout>
