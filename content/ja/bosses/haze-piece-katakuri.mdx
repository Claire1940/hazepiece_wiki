<Section icon={Sparkles} title="幻獣種モチモチの実の極意">

あらゆる **Haze Piece カタクリ** ビルドの核となるのはモチモチの実です。幻獣種クラスの実として、範囲攻撃（AoE）ダメージと速射攻撃に優れた多才なキットを提供します。特筆すべき機能の一つである「Rolling Donut」は、陸上と水上の両方を非常に高速で移動することを可能にし、クエストの完了やボス周回の速度を大幅に向上させます。

| 技名 | 説明 | 主な特徴 |
| :--- | :--- | :--- |
| **Power Mochi (力餅)** | 重い打撃攻撃 | 高いノックバック |
| **Grilled Mochi (焼き餅)** | 炎を纏ったパンチ | 継続的な燃焼ダメージ |
| **Dough Minigun (モチ機関銃)** | モチの弾丸を連射 | 照準・追跡可能 |
| **Rapid Punches (連撃)** | 高速の連続攻撃 | 単体DPSに最適 |
| **Ultimate Buzzcut (斬・餅)** | 大規模なAoE叩きつけ攻撃 | 強力なフィニッシャー |
| **Rolling Donuts (角餅)** | 高速移動形態 | 水上移動可能 |

これらの技を覇気と併用することが不可欠です。覇気が有効な状態ではモチモチの実のダメージ出力が大幅に増加し、ビッグ・マムのような高HPのボスを素早く削ることができます。

<Callout type="warning" title="スタミナ管理">
Dough MinigunとRapid Punchesはスタミナを大幅に消費します。戦闘中に圧力を維持し続けるために、エネルギー容量を増やすアクセサリーと組み合わせることをお勧めします。
</Callout>

</Section>

<Section icon={Target} title="鏡の世界への入場要件">

**Haze Piece カタクリ** の象徴的な武器を入手するには、鏡の世界に入る必要があります。これは簡単なタスクではなく、2つの異なる周回フェーズが含まれます。まず、ホールケーキアイランドでビッグ・マムに挑戦し、鏡の破片を入手する必要があります。このアイテムのドロップ率は極めて低いため、長時間のセッションを覚悟してください。

<Steps>
  <Step num="1" title="鏡の破片狩り">
    ホールケーキアイランドへ向かい、ボスであるビッグ・マムを倒します。鏡の破片は1%の確率でドロップします。入手後は、次のNPCまで運ぶ必要があるため、死なないように注意してください。
  </Step>
  <Step num="2" title="モチ島でSinを見つける">
    鏡の破片をモチ島にいるNPC「Sin」のところへ持っていきます。彼は破片を認識しますが、鏡の世界への入場を許可するために「貢ぎ物」を要求します。
  </Step>
  <Step num="3" title="ドーナツを10個集める">
    新しい島々（ホールケーキアイランドまたはモチ島）のNPCを狩ります。ドーナツを10個集める必要がありますが、これも通常のモブから1%の確率でドロップします。
  </Step>
  <Step num="4" title="ポータルを解放する">
    ドーナツを10個持ってSinの元へ戻ります。彼は鏡の世界のポータルを解放し、モチのボスのアリーナへのアクセスを許可してくれます。
  </Step>
</Steps>

| 要件 | 入手先 | ドロップ率 |
| :--- | :--- | :--- |
| **鏡の破片** | ビッグ・マム（ホールケーキアイランド） | 1% の確率 |
| **ドーナツ 10個** | 島のNPC | NPC1体につき1% |
| **50 ジェム** | クエスト報酬/デイリー | 確定 |
| **見聞色の覇気** | 覇気トレーナー | ボス戦に必須 |

<Callout type="success" title="周回戦略">
ドーナツ集めを加速させるには、複数のNPCを一箇所に引き寄せ、モチモチの実のAoE技（Ultimate Buzzcutなど）を使って同時に倒します。これにより、時間あたりのドロップチャンスを最大化できます。
</Callout>

</Section>

<Section icon={Swords} title="モチのボス（カタクリ）の攻略">

鏡の世界に入っても、すぐにボスが見つかるわけではありません。「Dough Fisher（スペースマン）」を見つけ、50ジェムを支払って「最強の戦士」――カタクリ自身を召喚する必要があります。このボス戦は **Haze Piece** の中で最も困難なものの一つであり、正確なタイミングと能動的な覇気管理が求められます。

モチのボスは、モチモチの実の技セットの強化版を使用し、モチの三叉戟による壊滅的なスキルも繰り出してきます。低体力のプレイヤーを一撃で倒しかねない「Ultimate Buzzcut」コンボに捕まらないよう、見聞色の覇気を常に有効にしておくことが極めて重要です。

| ボスの攻撃 | 対策 | 危険度 |
| :--- | :--- | :--- |
| **三叉戟スラム** | 横方向にダッシュ | 中 |
| **ラピッド・モチ** | 見聞色の覇気を発動 | 高 |
| **ドリルスラスト** | 距離を維持 | 中 |
| **バズカット・フィニッシャー** | 無敵フレームを利用 | 極限 |

ボスを倒すと、一定の確率でモチの三叉戟とカタクリのスカーフをドロップします。これらは確定ドロップではなく、同じサーバーで再びボスを召喚するには2時間のクールタイムがあることに注意してください。

<Callout type="info" title="ボスのクールタイム">
2時間の召喚タイマーはサーバー固有です。ジェムとドーナツの準備ができている場合は、サーバーホップ（サーバーの切り替え）を行うことで、すぐに新しいインスタンスでボスと戦うことができます。
</Callout>

</Section>

<Section icon={Shield} title="装備：三叉戟とスカーフのステータス">

**Haze Piece カタクリ** パズルの最後のピースは、モチの三叉戟とカタクリのスカーフです。これらのアイテムは、ゲーム終盤のPvPでモチモチの実を実用的なものにするために必要なステータスブーストを提供します。特に三叉戟は優れたリーチを誇り、敵の近接攻撃範囲外から攻撃を仕掛けることができます。

**モチの三叉戟のスキル:**
- **ドリルスラスト**: 前方に突進する攻撃で、敵のガードを貫通し、多段ヒットダメージを与えます。
- **三叉戟スラム**: 強力な振り下ろし攻撃で衝撃波を発生させ、周囲の敵をスタンさせます。

| アイテム | 主なバフ | 最適な用途 |
| :--- | :--- | :--- |
| **モチの三叉戟** | 剣ダメージ、リーチ | PvPコンボ |
| **カタクリのスカーフ** | 実ダメージ+15%、回避+2回 | 生存重視/ガラスキャノン |
| **モチモチの実** | 機動力、バースト | 総合的なゲームプレイ |

カタクリのスカーフは、現バージョンにおける実の能力者にとって間違いなく最高のアクセサリーです。追加の覇気回避は、ハイリスクな戦闘において重要な防御層となります。

<Callout type="tip" title="シナジーについて">
カタクリのスカーフによる実ダメージバフは覇気と重複するため、「Ultimate Buzzcut」はゲーム内で最高クラスのダメージを叩き出す技となります。
</Callout>
