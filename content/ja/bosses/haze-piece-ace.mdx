<Section icon={Map} title="フェーズ 1：インペルダウンでマッチを入手する">

エースを召喚するための旅は、恐ろしいインペルダウン刑務所から始まります。この厳重警備の場所には、召喚儀式の番人である署長（Warden）がいます。先に進むには、署長がマッチ（Match）をドロップするまで繰り返し倒す必要があります。

署長は中レベルのプレイヤーにとって手強い相手です。彼は高いダメージを与える近接攻撃と、準備不足のプレイヤーをすぐに圧倒する範囲攻撃を繰り出します。マッチのドロップ率が極端に低いため、このボスを周回することがプロセスの中で最も時間を要する部分です。

| ドロップアイテム | ドロップ率 | 用途 |
| :--- | :--- | :--- |
| **マッチ (Match)** | 1% | ジャングルのキャンプファイアに火を灯すために不可欠 |
| **署長の帽子** | 5% | 装飾品および僅かなステータスバフ |
| **ベリー (Beli)** | 100% | アップグレード用の標準通貨 |
| **経験値** | 100% | レベル上げに高い効率 |

<Callout type="warning" title="重要な警告">
マッチのドロップ率1%は、100回倒せば必ず手に入るという意味ではありません。300回以上の試行が必要だったという報告もあります。根気強さと素早いクリアタイムが不可欠です。
</Callout>

インペルダウンでの周回を最適化するには、署長をスタンさせることができる高DPSのビルドに焦点を当てましょう。グループでプレイする場合は、現在のサーバーの仕組みに応じて、最もダメージを与えたプレイヤー、あるいは最後の一撃を与えたプレイヤーがレアアイテムのドロップ率が高くなる可能性があることを全員で共有しておきましょう。マッチがインベントリに出現したら、次の場所へ運ぶ必要があるため、リセットしたりゲームを終了したりしないでください。

</Section>

<Section icon={Target} title="フェーズ 2：ジャングルの真珠パズル">

マッチを手に入れたら、ジャングルの島へ向かいます。この島は鬱蒼とした草木と中央の遺跡が特徴です。目的は、島の中央付近にあるキャンプファイアを見つけることです。

マッチを持った状態でこのキャンプファイアに干渉すると、儀式の次の段階が始まります。火を灯すと、4つの神秘的な真珠がジャングルのあちこちに散らばります。先に進むには4つすべてを見つけなければなりません。

<Steps>
  <Step num="1" title="キャンプファイアに火を灯す">
    マッチを装備し、ジャングルの遺跡にある火のついていないキャンプファイアを調べます。真珠が散らばったことを知らせるメッセージが表示されます。
  </Step>
  <Step num="2" title="周囲を探索する">
    真珠は通常、木の裏や島の岩壁の近くに隠されています。まずは北側と南側の海岸線をチェックしましょう。
  </Step>
  <Step num="3" title="4つの真珠をすべて集める">
    各真珠の上を歩いて回収します。これらは従来のインベントリスペースを占有しませんが、パズルのスクリプトによって追跡されます。
  </Step>
  <Step num="4" title="トーテムを起動する">
    エースが出現する祭壇の近くにある4つの小さな石のトーテムを見つけます。各トーテムに真珠を1つずつ配置して回路を完成させます。
  </Step>
</Steps>

<Callout type="success" title="召喚成功">
4つ目のトーテムが起動すると、画面が揺れ、祭壇に直ちに火拳のエースが出現します。戦闘の準備をしてください！
</Callout>

| 真珠の番号 | よくある場所のヒント |
| :--- | :--- |
| **真珠 1** | 海岸近くの大きな木の根元付近 |
| **真珠 2** | キャンプファイア近くの石柱の裏 |
| **真珠 3** | 高台の崖側にある茂みの中 |
| **真珠 4** | 入口付近のクエストNPCの近く |

</Section>

<Section icon={Swords} title="火拳のエースの戦闘戦略">

**Haze Pieceのエース**との戦いは、火傷スタックの管理と素早い飛び道具攻撃を回避する能力が試されます。エースは「火柱」や「火銃（Higan）」など、さまざまな火炎ベースの技を使用します。これらの攻撃は時間の経過とともに継続的なダメージを与えるため、体力の回復や防御シールドが不可欠です。

<CardGrid cols={2}>
  <Card title="遠距離戦略">
    - 火柱の範囲攻撃を避けるため、最大距離を保つ。
    - 氷やマグマの実を使用して、一貫した遠距離ダメージを与える。
    - エースがクールダウンのアニメーションに入っている間に削りダメージに集中する。
  </Card>
  <Card title="近接戦略">
    - エースが「火拳」のダッシュを使用した後にのみ攻撃を仕掛ける。
    - 覇気（Haki）を利用して、元素化による無効化を回避する。
    - 高い体力または「ライフリーチ（吸収）」アクセサリーを装備していることを確認する。
  </Card>
</CardGrid>

エースは通常の島のボスと比較して非常に高い体力を持っています。ソロで挑む場合、戦闘は数分間続くことを覚悟してください。戦闘中に死亡した場合、チームメイトがボスを戦闘状態に保っていない限り、召喚状態が失われる可能性があります。

<Callout type="info" title="報酬の分配">
エースを倒すと、十分なダメージを与えたすべてのプレイヤーが20ジェムを受け取ります。メラメラの実のドロップは個別の判定であり、非常に稀です。
</Callout>

| エースの技 | ダメージタイプ | 回避方法 |
| :--- | :--- | :--- |
| **火拳 (Fire Fist)** | 直線ダッシュ | 横にステップするか垂直にダッシュ |
| **火柱 (Fire Pillar)** | 円形AOE | ジャンプするか赤い円の外に留まる |
| **火銃 (Higan)** | 高速弾丸 | 動き続けるかガードする |
| **炎戒 (Enkai)** | 巨大爆発 | アリーナの端まで移動する |

</Section>
