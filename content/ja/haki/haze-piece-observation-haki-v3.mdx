アップグレードの流れはとても単純で、「レベルが高いほど、コストも高く、耐久性も上がる」という構造です。Observation Haki は一度有効化すると継続するので、本当に大事なのは、いつ切り替えるか、そしていつ回避を温存するかです。

| Tier | 主な効果 | 回避数 | 最適な用途 |
|---|---|---:|---|
| V1 | 基本的な敵感知 | 3 | 序盤の生存と移動 |
| V2 | 戦闘時の余裕が増える | 5 | 中盤の PvP と探索 |
| V3 | 最強の進行ティア | 7 | ボス戦、PvP、移動時の安全確保 |

| 要件 | 意味 | 実際の影響 |
|---|---|---|
| レベル 600 | V1 を解放 | 進行ラインを開始できる |
| レベル 1,500 | V2 を解放 | 回避数が増える中盤アップグレード |
| レベル 2,500 | V3 を解放 | 最も価値の高い最終ティア |
| Second Sea への到達 | V3 に必要 | 先に世界進行を終える必要がある |

3つのティアの捉え方としては、V1 がメカニクスを教え、V2 が戦闘を安定させ、V3 が安全余裕を最大化する、というイメージです。Beli が足りないなら、あまり防御に役立たない別アップグレードよりも、次のレベル条件を優先しましょう。

| 状況 | 役立つ理由 | 最適ティア |
|---|---|---|

| ボス狩り | 突発的な大ダメージを避けやすい | V3 |
| PvP の追跡戦 | 追加の回避で時間を稼げる | V2 または V3 |
| 狭いルート | 壁越しに圧力を把握できる | どの有効ティアでも可 |

<Section icon={Trophy} title="トラブルシューティングとFAQ">

<Callout type="info" title="基本的な対処法">
スキルが有効になっていないようなら、UI でティアを確認し、**R** を押したかを確かめ、回避がまだ残っているかを確認してください。
</Callout>

Observation Haki の問題の多くはバグではなく、進行ミスが原因です。必要レベルに達していない、順番を間違えて買った、V3 が Second Sea にあることを忘れている、というのがよくある原因です。

| 問題 | ありそうな原因 | 対処 |
|---|---|---|
| V1 を買えない | レベル 600 未満 | レベルを上げてから Marine Base Town に戻る |
| V2 を買えない | レベル 1,500 未満、または V1 未取得 | 先に V1 を完了してから Bubble Island に再訪する |
| V3 を買えない | Second Sea にいない、またはレベル 2,500 未満 | Second Sea に進み、レベル上げを続ける |
| 回避カウンターがない | Haki が有効化されていない | **R** を押して UI を確認する |
