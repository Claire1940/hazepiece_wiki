<Section icon={Target} title="ステータス配分とポイント上限">

2026年の最新アップデート時点で、1つのステータスに投資できる最大ポイント数は5250です。現在のレベルスケーリングでは、ユーティリティ用に少し余白を残しつつ、2つの主要ステータスを完全に最大化することができます。インペルジェイル（Impel Jail）や高レベル島のボス戦などのエンドゲームコンテンツに向けてビルドを最適化するには、これらのポイントの内訳を理解することが重要です。

| レベル範囲 | 合計ステータスポイント | カンスト可能な数 |
| :--- | :--- | :--- |
| **レベル 500** | 1,500 | 0（ハイブリッド期） |
| **レベル 1500** | 4,500 | 0（最初の上限間近） |
| **レベル 2500** | 7,500 | 1（特化期） |
| **レベル 3500+** | 10,500+ | 2（エンドゲーム最適化） |

ポイントの分配は、選択したプレイスタイルに基づいた優先順位に従うべきです。多くのプレイヤーは、「ガラスの砲台」のような実ビルドか、「高耐久」の剣士ビルドを選択します。どちらを選んだとしても、エリートNPCやボスにワンパンされないよう、「防御（Defense）」ステータスは常に共通の二次優先事項となります。

<Callout type="info" title="スタミナの最適化">
「格闘（Combat）」ステータスはスタミナを司ります。現在のメタでは、アクセサリーの助けなしでのスタミナ上限は250です。この上限に効率よく到達するには、格闘ステータスに**210ポイント**入れるだけで十分です。それ以上のポイントは、防御やメインのダメージステータスに回したほうが賢明です。
</Callout>

</Section>

<Section icon={Shield} title="ステータスのリセットと払い戻し方法">

特に序盤から中盤に移行する際、ステータス配分のミスはよくあることです。幸いなことに、Haze Pieceにはポイントを払い戻してキャラクターを再編するための方法がいくつか用意されています。最も確実な方法は特定のNPCを訪ねることですが、プロモーション報酬にも注目しておきましょう。

<Steps>
  <Step num="1" title="Spirit Mediumを見つける">
    **はじまりの島 (Starter Island)**へ向かいます。ここはすべてのプレイヤーが最初に訪れる島であり、キャラクター管理の拠点となります。
  </Step>
  <Step num="2" title="支払い方法を選択する">
    NPC「Spirit Medium」に話しかけます。**100ジェム（Gems）**または**55ロバックス（Robux）**を支払うことでステータスをリセットできます。無課金プレイヤーにはジェムが推奨されます。
  </Step>
  <Step num="3" title="有効なコードを使用する">
    公式DiscordやコミュニティWikiで「Stat Reset」コードをチェックしてください。これらはゲームのアップデートや節目に頻繁に配布され、無料でリセットが可能です。
  </Step>
  <Step num="4" title="タイムプレイギフトを確認する">
    コンパスの上のメニューを開き、紫色のパッケージアイコンを探します。ゲーム内でアクティブに過ごすことで「タイムプレイ」ギフトを獲得でき、たまにステータス払い戻しアイテムが含まれていることがあります。
  </Step>
</Steps>

| リセット方法 | コスト | 頻度 | 推奨プレイヤー |
| :--- | :--- | :--- | :--- |
| **Spirit Medium (ジェム)** | 100 ジェム | 無制限 | アクティブな農家 |
| **Spirit Medium (Robux)** | 55 Robux | 無制限 | 素早くリセットしたい人 |
| **プロモーションコード** | 無料 | 時々 | 全プレイヤー |
| **タイムプレイギフト** | 無料（時間） | 毎日/毎週 | 気長に待てる人 |

<Callout type="success" title="ジェム稼ぎ">
ジェムはデイリークエストの完了やボスの撃破で獲得できます。伝説級の実や強力な新しい剣を手に入れたときにビルドを変更できるよう、常に100ジェムは予備として持っておきましょう。
</Callout>

</Section>

<Section icon={Sparkles} title="レベル上げ戦略とステータスの節目">

フレンドとファーミングする際は、少なくとも一人が強力な範囲攻撃の実を持っていることを確認してください。NPCをまとめて引きつけ、同時に攻撃することで、クエストを数秒で完了できます。これはNPCのリスポーンが早い島で特に効果的です。

| ファーミング段階 | 推奨される島 | 主な目的 |
| :--- | :--- | :--- |
| **序盤 (Early)** | はじまりの島 / 海軍基地 | レベル200到達 / 刀の入手 |
| **中盤 (Mid)** | 空島 / 砂漠 | 範囲攻撃の実を確保 / 第1ステータスカンスト |
| **終盤 (Late)** | インペルジェイル / 新世界 | 第2ステータスカンスト / ボス狩り |

<Callout type="info" title="フレンドとのシナジー">
自分と異なるビルドタイプのフレンドと組むことで、弱点を補い合うことができます。防御重視のプレイヤーがボスの攻撃を受け止め（タンク）、実重視のプレイヤーが遠距離から強力な範囲ダメージを与えるといった連携が可能です。
</Callout>
